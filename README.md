
# Tour Ally

Tour Ally - A final year academic project, a tour guide application is to provide value
to travelers and enhance their travel experience.

## Features

- Weather Forecast
- My Planner
- Compass
- Currency Converter
- Language Translator
- Calendar
- Hire a Guide
- Tour Guideline Book
